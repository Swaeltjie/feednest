# FeedNest Keyboard Shortcuts

Press `?` at any time to open the keyboard hints overlay.

## Navigation

| Key | Action |
|-----|--------|
| `j` | Move selection down to next article |
| `k` | Move selection up to previous article |
| `Enter` | Open selected article in reading pane |
| `Escape` | Close reading pane / deselect |
| `g g` | Jump to first article |
| `G` | Jump to last article |
| `/` | Focus the search input |

## Article Actions

| Key | Action |
|-----|--------|
| `s` | Toggle star on selected article |
| `m` | Toggle read/unread on selected article |
| `d` | Dismiss selected article |

## Reading Pane

| Key | Action |
|-----|--------|
| `j` / `k` | Navigate to next/previous article (while reader is open) |
| `f` | Toggle focus mode (hides article list, reader goes full width) |
| `Escape` | Close reading pane and return to list |

## Views & Search

| Key | Action |
|-----|--------|
| `1` | Switch to hybrid (magazine) view |
| `2` | Switch to card grid view |
| `3` | Switch to list view |
| `v` | Cycle through view modes |
| `r` | Refresh feeds |

## Command Palette

| Key | Action |
|-----|--------|
| `Ctrl+K` / `Cmd+K` | Open command palette |
| Arrow keys | Navigate command palette results |
| `Enter` | Execute selected command |
| `Escape` | Close command palette |

## Help

| Key | Action |
|-----|--------|
| `?` | Toggle keyboard shortcuts overlay |
